Notable features



What is Authenticator Plus

Authenticator Plus is one-time passcode generator for 2-step verification, with Backup/Restore, Seamless Sync across devices, encrypted data storage and other customisation options.


Encryted data

Your account details are protected with 256-bit AES encryption, key is derived from your password using PBKDF2 with 64000 iterations. So even if your device is rooted your data is still protected.


Sync across devices

Account details are seamlessly synced between devices (Phone, Tablet, Kindle). Accounts can be edited offline, when device goes online, data will be synced autotmatically.


Backup / Restore

Backup / Restore all account details even if you are are using non-rooted device, so no worries if you are migrating to a new device.
